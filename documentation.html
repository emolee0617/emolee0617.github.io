Media/Image File

You are able to upload one or multiple media or image file(s) as the background. To upload file, click the the "choose file" button in the header menu. Feel free to choose one or more files. Noted that though you could upload the desired files in many formats, for your using experiences, uploading from `.jpg`, `.png`, `.jpeg` and `.mp4` formats are strongly suggested.

To delete one file, select the file on the bottom layer of timeline, and click the "Delete Media File" button in the header menu.

We also support to change the duration length for one file. To change the duration, click the file you want to make changes on the bottom layer of timeline. A "Bottom Layer Editor" will pop up. Input the new length for the file and click "Adjust Duration" Button. Units are in seconds.

Note that the default display duration for a image file is 10 seconds.
