Media/Image File

You are able to upload one or multiple media or image file(s) as the background. To upload a file, click the "choose file" button in the header menu. Feel free to choose one or more files. Note that though you could upload the desired files in many formats, for your using experiences, uploading from `.jpg`, `.png`, `.jpeg` and `.mp4` formats are strongly suggested.

To delete one file, select the file on the bottom layer of timeline, and click the "Delete Media File" button in the header menu.

We also support changing the duration length for one file. To change the duration, click the file you want to make changes on the bottom layer of the timeline. A "Bottom Layer Editor" will pop up. Input the new length for the file and click "Adjust Duration" Button. Units are in seconds.

Note that the default display duration for an image file is 10 seconds.

Save/Load Sessions

You are able to save and load presets of labels. To save a combination preset, click the "Session" button and choose the option "Save Session". The preset will be downloaded locally as an `.json` file.

To successfully upload a preset, click the "Session" button and choose the option "Upload Session". Choose the preset (in format of `.json` file) you want to upload and click open. Then, click the "Session" button again and choose the option "Upload Session Assets". Choose the directory containing the media files corresponding to the uploaded session and click open. It is important to upload the`.json` file first so that the assets get loaded in the correct order for the timeline.

Labels/Annotations

You are able to toggle On and Off of the Overlay layer, as well as create, delete edit labels on the Overlay layer.

To toggle the Overlay layer on the canvas board, click the "Toggle Overlay" button. You can also drag the overlay to the desired position on the canvas board by drag the red square on the top left corner of the overlay. To turn off the overlay, click the "Toggle Overlay" button again.

To create a new label, make sure the Overlay is on, and then click the Add label button in the top navbar. Then click the position where you want to put the label on the Overlay. A new label will be created.

When you click the label from the Overlay or from the Timeline, the Label Editor will show up. You are able to edit the label's text, font, and background color, upload a small image or gif file to the label, change position on Overlay, resize, and change the start and end time of that label. Besides, you can also drag the label box on the timeline to change its start and end time.

To delete one label, after selecting that label, click the "Delete Label" button in the Label Editor.

Play with Timeline

You are able to play and pause the media you uploaded, by clicking the "Play" and "Pause" button on the timeline. You can also drag the timeline slider (the grass-green color triangle) to move to a certain timestamp. When creating new labels and uploading new media files, the default start time will be where the timeline slider is. You are able to drag the label boxes and media file boxes on Timeline to adjust new start and end time.
